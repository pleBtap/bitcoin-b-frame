## What is this?

Bitcoin B Frame is a **standing frame in the shape of the Bitcoin logo** (the iconic "B"). It's designed to hold photos, cards, notes, or just look great on your desk.

The model is fully parametric — change dimensions, proportions, and features to match your needs.

## File Formats

| File | Description |
|------|-------------|
| `bitcoin-b-frame.FCStd` | FreeCAD source file — fully parametric |
| `bitcoin-b-frame.step` | STEP file — universal CAD exchange format |

## Customization

Open `bitcoin-b-frame.FCStd` in [FreeCAD](https://www.freecad.org/) and adjust:

- **Frame height/width** — overall size
- **Slot width** — how thick an object the frame can hold
- **Base depth** — stability vs. footprint
- **Logo proportions** — tweak the iconic "B" shape
