# Contributing to Bitcoin B Frame

Thanks for your interest in contributing! 🚀

## How to Contribute

### Reporting Issues

Found a bug or have a suggestion? Open an issue with:
- Clear description of the problem or idea
- FreeCAD version (if applicable)
- Screenshots if helpful

### Pull Requests

1. Fork the repository
2. Create a branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Test in FreeCAD (open/save/export works correctly)
5. Commit with clear messages
6. Push and open a Pull Request

### What We're Looking For

- ✅ Improved parametric features
- ✅ Better FreeCAD spreadsheet integration
- ✅ Additional export formats (STL variants, OBJ, 3MF)
- ✅ Documentation improvements
- ✅ Photos of printed/milled results
- ✅ Parameter presets (e.g., "photo frame", "business card holder", "wallet stand")

### Design Guidelines

- Keep the Bitcoin "B" recognizable
- Ensure the frame stands stable without supports
- Maintain parametric freedom (avoid hard-coded dimensions)
- Document any new parameters in the FreeCAD spreadsheet

## Code of Conduct

Be respectful and constructive. We're all here to build cool stuff.

## Questions?

Open a discussion or reach out via the repository's communication channels.
